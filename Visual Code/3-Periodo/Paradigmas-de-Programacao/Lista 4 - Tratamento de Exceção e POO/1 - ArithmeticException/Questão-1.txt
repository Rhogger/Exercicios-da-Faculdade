Crie um metodo chamado realizarDivisao, que deverá receber 2 paramentros inteiros (A e B). Ao final, retornar a divisão entre B e A (B / A). O programa deverá para quando o usuario informar o valor 0 para A e 0 para B. Adicione o tratamento ArithmeticException() para informar qualquer tipo de erro aritmético que possa existir. Caso não seja uma exceção aritmética, utilizar a exceção Exception() para informar qualquer outro erro que possa ter ocorrido
